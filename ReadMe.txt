Step1 : Download Node.js from https://nodejs.dev/en/download/
Step2 : Install Node.js globally in your system.
Step3 : Create database "test" in mysql.
Step4 : Set mysql username, password and host in .env file.
Step3 : Open project directory in node.js command prompt.
Step4 : Run "npm i" command to install node modules as per package.json file.
Step5 : Run "npm start" to run the project.
Step6 : Tables created when ou run the proect.
Step7 : Check api from postman.
